# Pokémon Data Dashboard (R)

This project is an interactive dashboard built using R, flexdashboard, and plotly to explore Pokémon statistics.

## Features
- Interactive filtering by Pokémon type
- Dynamic visualizations using plotly
- Scatter plots and box plots to analyze relationships between stats
- Clean dashboard layout for data exploration

## Tools Used
- R
- ggplot2
- plotly
- flexdashboard
- tidyverse

## Files
- `.Rmd` file contains the dashboard code
- `.html` file contains the rendered interactive dashboard
- `pokemon.csv` dataset used for analysis

## Purpose
This project demonstrates data visualization and dashboard development using R.
